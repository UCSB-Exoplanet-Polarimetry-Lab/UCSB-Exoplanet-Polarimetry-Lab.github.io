As part of my research I have led the development of and contributed to a range of projects, including:

  • The GPI Data Reduction Pipeline - The main data reduction pipeline for the Gemini Planet Imager
  • The WIRC+Pol Data Reduction Pipeline - The main data reduction pipeline for WIRC+Pol
  • PSISIM - A tool to simulate science outputs from high-contrast imagers, spectrographs and polarimeters
  • pyMuellerMat - A library to generate and work with flexible Mueller Matrices for simulating polarimeters
  • pyklip - A post-processing library for high-contrast data
Please check out my Github Page for my recent activity!
